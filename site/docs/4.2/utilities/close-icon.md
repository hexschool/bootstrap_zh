**確保為螢幕閱讀器添加文字**(讓有閱讀障礙者也能順利閱讀網頁)，我們使用 `aria-label` 標籤屬性。

{% capture example %}
<button type="button" class="close" aria-label="Close">
  <span aria-hidden="true">&times;</span>
</button>
{% endcapture %}
